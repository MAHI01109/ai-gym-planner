# 🏋️ AI Gym Workout Planner

AI-powered gym workout planner built with modern web technologies.

## 🚀 Features

* AI workout plan generation
* Personalized fitness plans
* Clean modern UI
* Responsive design
* Download workout plan

## 🛠 Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* shadcn/ui
* OpenAI API

## 📦 Installation

```bash
git clone https://github.com/yourusername/ai-gym-planner.git
cd ai-gym-planner
npm install
npm run dev
```
